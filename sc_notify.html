sc_notify

ESX QBCore Standalone

A simple notification script for displaying notifications. Customizable.

Configuration

  • title="Custom-Notify"
  • description="With your own icon and position!"
  • type=info -- success, error, info, warning
  • icon=FontAwesome (z.B. fa-solid fa-bell)
  • iconColor=#FF00FF
  • position="top-right" -- top, top-right, top-left, bottom, ...
  • Duration=15000
  • showDuration=false

Default

exports['sc_notify']:notify({
    title = "Standard-Notify", 
    description = "This is a standard info!",
    type = "info"
})
TriggerClientEvent('sc_notify:notify', targetPlayerId, {
    title = "Standard-Notify", 
    description = "This is a standard info!",
    type = "info"
})

Coustom

exports['sc_notify']:notify({
    title = "Custom-Notify", 
    description = "With custom icon & position!",
    icon = "rocket",
    iconColor = "#FF00FF",
    position = "top-right",
    Duration = 15000,
    showDuration = false
})
TriggerClientEvent('sc_notify:notify', targetPlayerId, {

    description = "With custom icon & position!",

    iconColor = "#FF00FF",
    position = "top-right",
