Introduction

ScubeScripts is a collective of designers and developers building modular, NUI-ready scripts for FiveM. Our goal is to ship maintainable, secure and performant systems that are easy to integrate into your server.

This documentation contains setup instructions, API references and examples for all official ScubeScripts projects. Each project page includes installation steps, configuration options and quick usage snippets so you can get running fast.

Need help? Start with the Installation guide, or browse the Examples section for ready-to-use snippets and demos.
